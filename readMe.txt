1.В папке конфиг должны быть 2 файла:

    default.json с содержимым:
    {
    "TELEGRAM_TOKEN": "ключ",
    "OPENAI_KEY": "ключ",
    "TEST_ENV": "dev"
    }

    и

    production.json с содержимым:
    {
    "TELEGRAM_TOKEN": "ключ",
    "OPENAI_KEY": "ключ",
    "TEST_ENV": "prod"
    }

    TELEGRAM_TOKEN - токен телеграмм бота
    OPENAI_KEY - токен ChatGPT
2. npm init
